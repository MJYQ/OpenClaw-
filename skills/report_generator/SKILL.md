# Report Generator Skill

## Skill名称

招聘报告生成 Skill（Report Generator Skill）


## Skill描述

该Skill用于汇总招聘过程中的候选人数据，
自动生成招聘分析报告，
帮助企业管理人员了解招聘情况。


## Skill目标

实现招聘数据自动整理，
减少人工统计工作。


## 功能说明


主要功能：

1. 汇总候选人信息

2. 统计招聘数据

3. 生成Markdown报告

4. 输出招聘分析结果
